# Bee Well: Return to Center

A neuro-affective stabilization tool designed for individuals navigating ADHD, BPD, CPTSD, GAD, and MDD.

## The Architecture
Based on the "State → Fix → Step" protocol:
1. **Identify State**: Choose the color matching your current internal weather.
2. **Apply Fix**: Perform the immediate physiological regulation task.
3. **Execute Step**: Perform the Minimum Viable Shift (MVS).

## Tech Stack
- **React + Vite**: High-performance UI.
- **Tailwind CSS**: Low-stimulation, high-utility styling.
- **Zustand**: Localized "Memory Shield" ledger (No data leaves the device).
- **PWA**: Fully installable and offline-capable.
